Sobre o Projeto:

Essas informações tiveram como base a pesquisa feita pelo Detran, no
ano de 2020. Todo o tratamento de dados e análise foi feito pela equipe
de Desenvolvedores: Mylena Santos, Ewergthon Renan, Emilly
Silva e Samara Beatriz. Na base de dados do Detran, referente ao ano de
2020,temos as informações de todos os acidentes automobilísticos regis-
trados no Brasil. A partir disso realizamos todas as etapas da análise exploratória
dos dados para realizar esse estudo.

Sobre a base:

Esta base de dados é disponibilizada pelo Detran (Departamento Estadual de Trânsito).
Nela, há informações sobre todos os acidentes de carro registrados no Brasil no ano de 2020.
A base contém uma grande quantidade de dados que possibilitam a realização de diversos estudos
valiosos.

Atributos da Base:
  • Num_boletim: Número de registro do acidente
  • data_hora_boletim: Data e hora do registro
  • Nº_envolvido: Número de pessoas envolvidas no acidente
  • condutor: Se o condutor teve responsabilidade no acidente
  • desc_severidade: Se o acidente foi faltal ou não
  • sexo: Sexo da pessoa
  • cinto_seguranca:Se eles usavam cinto de segurança
  • Embreagues: Se o condutor esta embreagado
  • Idade: Idade
  • nascimento: Data de nascimento
  • categoria_habilitacao: Tipo de habilitação
  • descricao_habilitacao: Descrição ou qualificação da carteira do motorista
  • declaracao_obito: Se houve alguma morte ou não
  • cod_severidade_antiga: código de severidade do registro
  • especie_veiculo: Tipo do veículo
  • pedestre: Se tinha algum pedestre envolvido no acidente
  • passageiro: Se tinha passageiro no veículo
